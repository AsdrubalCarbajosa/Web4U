HTML

The standard markup language for Web pages

For this tutorial, we're gonna be seeing three very useful tags for anyone who wants to get started with HTML.

Headings

Tags go from <h1>(Most important) to <h6>(Less important), with each value defining how important the information you want to share is.
This is useful to present things in a way that is visually more pleasant than just writing it using <p>.


If you want to learn more, you can go here.

Images

You can insert images into your website using the tag <img>, along with src, which allows you to specify the path of the image.


If you want to learn more, you can go here.

Tables

To create a tale in HTML, first you need to declare it with the tag <image>, then separate each row with <tr>, and after that you separate
each column with either <td> (if you want it to be centered), or <th> (if you want it to be left sided).


If you want to learn more, you can go here.
