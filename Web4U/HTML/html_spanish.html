HTML

El lenguaje de programación estándar para páginas web.

Para este tutorial, veremos tres etiquetas muy útiles para cualquiera que quiera empezar con HTML.

Encabezados

Las etiquetas van desde <h1> (más importante) a <h6> (menos importante), con cada valor defininiendo qué tan importante es la información que se desea compartir.
Esto es útil para presentar la información de una manera visualmente más agradable que simplemente escribiendolo con <p>.


Si quieres saber más, puedes ir aquí.

Imágenes

Puedes insertar imágenes en tu sitio web mediante la etiqueta <img>, junto con src, que te permite especificar la ruta de la imagen.


Si quieres saber más, puedes ir aquí.

Tablas

Para crear una tabla en HTML, primero debes declararla con la etiqueta <image>, luego separar cada fila con <tr> y, a continuación, separar
cada columna con <td> (si quieres que quede centrado), o <th> (si quieres que quede del lado izquierdo).
