CSS

The markup language used to style HTML

For this tutorial, we're gonna be seeing three very useful tags for anyone who wants to get started with CSS.

Padding

The tag <padding> is used for establishing the space you want at the side of an image, text, etc.
It is often used in four ways: <padding-top>, <padding-left>, <padding-right> and <padding-bottom>.

If you want to learn more, you can go here.

Height and Width

The height and width properties are used to set the height and width of an element.
The height and width properties do not include padding, borders, or margins. It sets the height/width of the area inside the padding, border, and margin of the element.


If you want to learn more, you can go here.

Lists

In HTML, there are two main types of lists: unordered lists (<ul>), where the list items are marked with bullets and ordered lists (<ol>), where the list items are marked with numbers or letters
