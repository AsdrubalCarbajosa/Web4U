CSS

The markup language used to style HTML

For this tutorial, we're gonna be seeing three very useful tags for anyone who wants to get started with CSS.

Padding

The tag <padding> is used for establishing the space you want at the side of an image, text, etc.
It is often used in four ways: <padding-top>, <padding-left>, <padding-right> and <padding-bottom>.

If you want to learn more, you can go here.

Images

You can insert images into your website using the tag <img>, along with src, which allows you to specify the path of the image.


If you want to learn more, you can go here.

Tables

To create a tale in HTML, first you need to declare it with the tag <image>, then separate each row with <tr>, and after that you separate
each column with either <td> (if you want it to be centered), or <th> (if you want it to be left sided).


If you want to learn more, you can go here.
